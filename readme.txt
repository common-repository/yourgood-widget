== Description ==

**Yourgood.Widget сделает посетителей сайта лидами**

**Yourgood.Widget** – бесплатная кнопка ля связи через популярные мессенджеры для сайта. Предложите клиентам удобный способ связи и получайте больше лидов с сайта. Установка, настройка дизайна и расположения кнопки — 5 минут.

**Достает для вас самое ценное — номер телефона клиента**

Сможете связаться с клиентом, даже когда он ушел с сайта. Номер телефона — это возможность звонить, писать в мессенджеры, отправлять рассылки и напоминать о себе

**Собирает лидов лучше, чем онлайн-чат, обратный звонок и форма обратной связи**

Чтобы связаться с вами, человеку не нужно заполнять формы, ждать обратного звонка и выбирать, где написать. Он просто пишет из любимого мессенджера

**Подходит для сквозной аналитики, мэчится с системами учета и аналитики**

Показываем источник перехода, конверсию в клик и диалог. Вы сможете подключить интеграцию с amoCRM, Битрикс24, Wazzup, Яндекс.Метрика, Google Analytics, UIS, Calltouch и Roistat

**Позволяет клиенту легко писать вам с компьютера и телефона**

Если клиент смотрит сайт с телефона — перейдет в чат по кнопке. Если с компьютера — по QR в приложение на телефоне или по кнопке — в веб-версию

**Партнерская программа**

Если вы делаете сайты для клиентов или интегрируете CRM — пишите в [телеграм] (https://t.me/yourgood_widget_bot). Обсудим партнёрские плюшки и будем сотрудничать.

**Использование стороннего программного обеспечения**

Плагин Yourgood.Widget использует стороннюю компанию [Yourgood.Widget] (https://www.widget.yourgood.ru) для сбора, обработки и отправки сквозной аналитики в CRM или аналитические системы пользователей.
Более подробную информацию можно найти в [Условиях использования] (https://www.widget.yourgood.ru/termsofuse/) и [Политике конфиденциальности] (https://widget.yourgood.ru/wordpresspluginpolicy).

== Installation ==

Из панели WordPress

1. **Зайдите** Плагины > Добавить новый
2. **В поиске** вбить "yourgood-widget"
3. **Установка и активация** нажмите кнопку "Установить", затем на странице плагинов активируйте плагин Yourgood.Widget
4. **Зарегистрируйтесь** в [Yourgood.Widget](https://widget.yourgood.app/sign-in)
5. **В личном кабинете** [Yourgood.Widget](https://widget.yourgood.app/sign-in) в разделе "Каналы" добавьте номер WABA и/или Telegram
6. **Перейдите** в раздел "Интеграции" личного кабинета и скопируйте уникальный ID код Wordpress
7. **Вставьте** уникальный ID код Wordpress в поле ниже и нажмите "Сохранить"
8. **Остались вопросы?** Перейдите на [сайт](https://yourgood.ru/) для уточнения информации

1. Бесплатная кнопка перенаправит лида с сайта в мессенджер — расположение и размер кнопки настраивается

2. Симулятор мессенджера и всплывающее сообщение привлекут внимание посетителей сайта и активирует молчаливых наблюдателей

3. Ловец лидов покажет ваше предложение посетителям сайта. Всплывашка появится по триггеру и подтолкнет к действию

4. Yourgood.Widget собирает статистику по обращениям. В личном кабинете вы узнаете: количество посетителей, конверсию в клик и диалог
